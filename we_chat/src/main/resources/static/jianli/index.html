平安VIP项目

开发时间：2016 /10--2019.3

相关技术：sqoop + oracle + Flume + Kafka + hive + spark + redis + HDFS+kettle

本系统主要的功能在于通过同步平安旗下寿险、产险、养老险、证券、信托、银行、基金等专业公司的VIP客户以及银行大零售用户和寿险高价值用户到权益系统数据平台，对这些客户按照一定的规则进行建档定级处理，并赠送积分以及各种权益从而维持客户的忠诚度 权益系统数据对应平台应用系统主要包括：
1.MVIP系统： 提供VIP俱乐部H5页面，主要功能包含生日礼物选择，高价值抽奖活动，积分产品兑换，礼遇查询、礼遇申请、会员信息查询、订单查询等功能。应用于财富宝APP、微信公众号、金管家APP、口袋E等。
2.VIP后台管理系统主要对各专业公司机构人员、会员信息、增值服务订单、业务数据报表、活动监控报表等进行管理

主要职责：
1.通过flume对接MVIP核心子系统进行日志收集并写入HDFS通过，spark进行指标分析、统计，主要维度包括时间维度、浏览器维度、平台维度、人员类型维度，层级维度等，把分析的结果保存到Oracle 中以供后台管理系统数据报表展示使用
2.通过kettle编写定时job同步第三方数据，或者将数据同步给第三方关联系统
3.使用sqoop将VIP服务订单数据从oracle 迁移到 hdfs，统计VIP服务监测报表结果，并回写到Oracle数据
4.通过hive统计高价值用户人数和属性分布结果，按区域维度，年龄段维度，层级维度，星座维度，保单维度等进行计算分析
5.业务数据对账，报表统计，结果入库，实时告警
6. 编写hive 和 spark 对用户数据离线处理进行建档，定级，送分等处理，并同步到oracle
